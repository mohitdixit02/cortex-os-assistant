## CORTEX
This Module consists of the all Models responsible for performing the core functions:

#### Main
Main Class handles the main loop of the program, it is responsible for calling the other classes and functions in the correct order.

#### Memory
Memory Class is responsible for storing and retrieving information. It can be used to store information about the user, the environment, and the tasks that need to be performed.

#### Voice
Voice Class is responsible for handling all voice-related functions, such as text-to-speech and speech-to-text. It can be used to communicate with the user and to receive commands.

#### Manager
Manager Class is responsible for managing the tasks that need to be performed. It can be used to create, update, and delete tasks, as well as to track their progress.
